# Phishing-Attack-Simulation-Using-GoPhish
Phishing simulation project using GoPhish to demonstrate credential harvesting in a controlled environment
Phishing Attack Simulation using GoPhish

> Overview

This project demonstrates a phishing attack simulation using GoPhish in a controlled environment to understand how attackers capture user credentials.

> Tools Used

* GoPhish
* Kali Linux
* Gmail SMTP

> Methodology

* Created phishing email template
* Designed fake login landing page
* Configured SMTP for email delivery
* Launched phishing campaign
* Collected and analyzed results

> Results

* Emails Sent: 1
* Link Clicked: Yes
* Credentials Captured: Yes

> Ethical Considerations

This project was conducted in a controlled environment for educational purposes only. No real users were targeted.


> Limitations

* SMTP restrictions
* Local network dependency

> Future Improvements

* Implement MFA
* Improve email filtering
* Conduct awareness training

> Conclusion

Phishing attacks exploit human trust. This simulation highlights the importance of cybersecurity awareness and preventive measures.
